# Sales-Analysis-Dashboard

#Project Objective

The E-Commerce Sales Performance Dashboard is developed to analyze and visualize sales data to support business decision-making. The dashboard provides insights into sales performance, profit analysis, customer segmentation, regional performance, and product category analysis

#Dataset Used

The dataset includes order details, customer information, product categories, sales amount, quantity, discount, profit, and regional data.
